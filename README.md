# KeyBrowser
#### キーボードによる快適な操作を"目指した"ブラウザ
!["screenshot"](https://gyazo.com/887b5216302d6109d93b6ba6caaa0ba5.png)


## 開発環境
* Electron
* React
* MobX
* FuseBox
* (TypeScript)


## 実行方法
1 パッケージのインストール

```bash
yarn # or npm i
```

2 ビルド

```bash
make dev # production -> make prod
```

3 起動

```bash
make start
```


## 使い方
- ⌘K → 検索
